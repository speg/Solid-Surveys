Welcome to SolidSurveys.

SolidSurveys is a API demo for FluidSurveys. It is a minimalist, lightweight, and simple to use web app. It's text only interface makes it super simple to use, offers faster performance, and has greater browser and accessibility support.

CoffeeScript

SolidSurveys was also a first exercise in learning CoffeeScript.

FluidSurveys API

Learn more about the FluidSurveys API at http://docs.fluidsurveys.com/
